# Toàn bộ quy trình Certificate → Flash → Xác thực Sidewalk

> **Note:** This document was generated by Claude (Anthropic AI) based on analysis of the STM32-Sidewalk-SDK source code. Generated on 2026-03-16.

---

## 1. Provision Tool chuyển JSON → HEX

```
certificate.json → provision.py → st_aws_WBA55xG.hex
```

Tool đọc JSON chứa keys/certificates từ AWS, encode theo format **TLV (Tag-Length-Value)** rồi ghi vào file `.hex` với địa chỉ flash cố định.

## 2. Vùng nhớ Flash (Memory Map)

Từ linker script `STM32WBA5xxG_Sidewalk.ld`:

```
┌──────────────────────────┐ 0x080FFFFF
│   MFG_DATA (8 KB)        │ ← Certificate ở đây
│   Addr: 0x080FE000       │
├──────────────────────────┤ 0x080FDFFF
│                          │
│   Application Code       │ ← Firmware sid_900
│   (Phần còn lại Flash)   │
│                          │
└──────────────────────────┘ 0x08000000
```

## 3. Dữ liệu Certificate lưu trong MFG_DATA

Format TLV (Version 8), mỗi entry gồm `[Tag 2B | Length 2B | Value NB]`, căn chỉnh 16-byte:

```
MFG_DATA Region (8 KB @ 0x080FE000):
┌─────────────────────────────────────────────────┐
│ Version (4 bytes) = 0x00000008 (TLV v8)         │
├─────────────────────────────────────────────────┤
│ [Tag=1]  DEVID (5 bytes) - Device ID            │
│ [Tag=3]  SERIAL_NUM (17 bytes) - SMSN           │
├─────────────────────────────────────────────────┤
│ [Tag=6]  DEVICE_PRIV_ED25519 (32 bytes)         │ ← Private key ký xác thực
│ [Tag=7]  DEVICE_PUB_ED25519 (32 bytes)          │ ← Public key
│ [Tag=8]  DEVICE_PUB_ED25519_SIGNATURE (64 bytes)│ ← Chữ ký của public key
├─────────────────────────────────────────────────┤
│ [Tag=9]  DEVICE_PRIV_P256R1 (32 bytes)          │ ← Private key (ECDSA)
│ [Tag=10] DEVICE_PUB_P256R1 (64 bytes)           │
│ [Tag=11] DEVICE_PUB_P256R1_SIGNATURE (64 bytes) │
├─────────────────────────────────────────────────┤
│ [Tag=18] PRODUCT_PUB_ED25519 (32 bytes)         │ ← Certificate chain
│ [Tag=19] PRODUCT_PUB_ED25519_SIGNATURE (64B)    │     ↓
│ [Tag=24] MAN_PUB_ED25519 (32 bytes)             │   Manufacturer
│ [Tag=25] MAN_PUB_ED25519_SIGNATURE (64 bytes)   │     ↓
│ [Tag=30] SW_PUB_ED25519 (32 bytes)              │   Sidewalk
│ [Tag=31] SW_PUB_ED25519_SIGNATURE (64 bytes)    │     ↓
│ [Tag=36] AMZN_PUB_ED25519 (32 bytes)            │   Amazon Root CA
│ [Tag=37] AMZN_PUB_P256R1 (64 bytes)             │
└─────────────────────────────────────────────────┘
```

```

                                    │
                              ┌─────┴──────┐
                              │ Đọc version│
                              │ @ 0x080FE000│
                              └─────┬──────┘
                                    │
                         Version == 8 (TLV)?
                        ┌───YES──┤    ├──NO──┐

                   index        (Fixed offset)         → System halted

```

         │                                    │                              │
    ③ ──── BLE Advertisement ────────────>    │                              │
         │                                    │                              │
    ④ <──── Connection Request ──────────     │                              │
         │                                    │                              │
    ⑤ Đọc DEVICE_PRIV_ED25519 từ flash       │                              │
       Ký authentication message              │                              │
         │                                    │                              │
    ⑥ ── Signed Auth Message ────────────>    │                              │
         │                              Forward──────────────────────>       │

         │                                    │                      bằng Device Public Key

## 6. Chi tiết xác thực (Certificate Chain)

```
Amazon Root CA (AMZN_PUB_ED25519)        ← Tin cậy tuyệt đối
       │ ký
Sidewalk Public Key (SW_PUB_ED25519)     ← Cấp bởi Amazon
       │ ký
Manufacturer Key (MAN_PUB_ED25519)       ← Cấp bởi Sidewalk (cho ST)
       │ ký
Product Key (PRODUCT_PUB_ED25519)        ← Cấp bởi Manufacturer
       │ ký
Device Public Key (DEVICE_PUB_ED25519)   ← Key của thiết bị cụ thể
       │
Device Private Key (DEVICE_PRIV_ED25519) ← Bí mật, chỉ nằm trên chip
       │
       └── Dùng để KÝ mọi message gửi đi
```

AWS Cloud xác minh ngược từ **Device → Product → Manufacturer → Sidewalk → Amazon Root** để đảm bảo thiết bị là hợp lệ.

## 7. Tóm tắt

| Bước | Mô tả | File liên quan |
|------|--------|----------------|
| JSON → HEX | Provision tool encode keys theo TLV format | `tools/provision/provision.py` |
| Flash | Ghi vào vùng MFG_DATA (0x080FE000, 8KB) | Linker script `.ld` |
| Boot | `sid_pal_mfg_store_init()` validate + build index | `pal/.../mfg_store.c` |
| Auth | Đọc private key → ký message → gửi qua gateway → AWS verify certificate chain | Sidewalk SDK prebuilt libs |
