# Release Notes for lib_gnss Library

## Purpose
The **lib_gnss** Library provides the high-level API of the lib_gnss library. It is a complete middleware to build applications using the Teseo-LIV3F, Teseo-VIC3DA, and Teseo-LIV4F GNSS chips.
